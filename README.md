COMPETITIVE CODING TRACKER
A frontend-based Competitive Programming Tracker built using HTML, CSS, and JavaScript to help users manage daily practice, revision problems, and coding contests â€” with all data stored locally using localStorage.

FEATURES
   - DASHBOARD
        Total problems solved
        Problems pending for revision
        Problems stored for revision
        Upcoming contest countdown
        Last revised date

   -TRACKER
        Add daily coding tasks
        Mark problems as Done
        Automatically move completed problems to a history table
        Track number of problems solved per day

   -CONTEST
        Add upcoming contests with date, time, and platform
        Automatically moves contests to Past Contests once completed

   -REVISION
        Create topic-wise revision lists
        Add problem links under each topic
        Mark problems as revised
        Revised problems move to a revision history table with date

TECH STACK
    -HTML - layout
    -CSS - style 
    -JAVASCRIPT- logic
    -LOCALSTORAGE - data persistence(no backend)

REASONS FOR MAKING THIS PROJECT
    - Practice core JavaScript concepts (DOM, events, arrays, localStorage)
    - Build a real-world productivity tool for competitive programming
    - Avoid backend complexity while still managing persistent data

STEPS TO RUN : 
    - Clone the repository
    - Open using live Server
    - Start Tracking your progress
